## 一、核心理念

### 1.1 游戏调性

**「在平凡的校园日常里，发现那些让心跳加速的微小瞬间。」**

Campus Heartbeat 不追求戏剧化的冲突与反转，而是聚焦于：
- 🌸 **细腻的日常互动** — 帮忙捡文件、一起看晚霞、操场边递饮料
- 💭 **渐进的情感升温** — 从陌生到熟悉，从好奇到心动，从暧昧到告白
- 🎭 **真实的性格展现** — 每个角色在不同阶段展露不同层次的性格
- 📷 **记忆定格的仪式感** — CG 不只是奖励，是两人共同经历的浓缩

### 1.2 玩家角色定位

玩家扮演一名 **刚转学到晨曦学院的高二男生**，性格温和但不软弱，善于观察却不主动出击。转学的原因是父亲工作调动，对新环境既期待又有些不安。

**设计原则**：
- 玩家角色不是白纸——他有基本的人设和情感反应
- 但选择权归玩家——每次互动至少提供 2 个有意义的选择
- 不存在「正确答案」——不同选择通向不同的关系质感，不是简单的好坏

### 1.3 叙事节奏公式

```
每条角色路线 = 5 幕结构
  第一幕「偶遇」 — 建立第一印象，展现角色标志性特质
  第二幕「靠近」 — 找到共同话题/合作契机，关系自然升温
  第三幕「波澜」 — 出现小矛盾或误解，考验双方心意
  第四幕「破冰」 — 化解隔阂，关系质变
  第五幕「定格」 — 情感确认，解锁结局 CG
```

## 二、世界观

**全名**：私立晨曦综合学院（Chenxi Academy）

**定位**：一所位于海滨城市的中等规模私立高中，以社团文化活跃著称。学校不追求升学率排名，而是鼓励学生发展多元兴趣，因此诞生了大量风格各异的社团。

**校训**：「让每一天的日出都值得记住。」

## 四、时间与空间系统

### 4.1 时间流转规则

```
一天 = 3 个时段
  ┌──────────┐   ┌──────────┐   ┌──────────┐
  │  早晨     │ → │  放学后   │ → │  夜晚    │ → (次日早晨)
  │ morning   │   │after_school│  │  night   │
  │ 7:00-12:00│   │14:00-18:00│   │19:00-22:00│
  └──────────┘   └──────────┘   └──────────┘
```

**时间推进条件**：
- 触发剧情事件（大部分事件会推进时间）
- 主动选择「休息/前往下一时段」（探索后无事件时）
- 某些日常事件不推进时间（标记为 `advances_time: false`）

### 4.2 每日行程模板

| 时段 | 玩家典型行为 | 可遇角色 |
|------|------------|---------|
| **早晨** | 从校门进入校园，经过教学楼前。上课时间不可自由移动。 | 林澄（校门检查值日）|
| **放学后** | 自由探索校园各地点。社团活动时间。 | 林澄（操场/社团楼）、雪纪（教学楼前/中庭/码头）|
| **夜晚** | 校园人少，氛围变化。部分地点不可进入（社团楼关门）。 | 林澄（天台）、雪纪（校门/灯塔）|

### 4.3 天数与剧情节奏

游戏设计为 **28 天**（4 周），每周有自然的叙事节奏：

| 周次 | Day 范围 | 叙事节奏 | 核心事件密度 |
|------|---------|---------|------------|
| **Week 1** | Day 1-7 | 相遇与好奇 | 高（初遇+试探） |
| **Week 2** | Day 8-14 | 靠近与日常 | 中（中期事件+日常积累） |
| **Week 3** | Day 15-21 | 波澜与考验 | 高（矛盾+化解） |
| **Week 4** | Day 22-28 | 定格与告别 | 高（结局事件） |

## 五、主线剧情大纲

### 5.1 故事梗概

**九月初，陆辰转学到海滨城市的晨曦学院。**

在这所以社团文化著称的学校里，他遇到了两个截然不同的女孩——

一个总是独自扛起所有责任，在校门口被风吹乱文件也不愿开口求助的学生会副会长 **林澄**；

一个用相机代替语言，在教学楼前半蹲着拍晚霞却突然把镜头转向自己的摄影社独行侠 **雪纪**。

在日复一日的校园生活中，陆辰从旁观者逐渐走进她们的世界——帮林澄整理堆积如山的社团文件，做雪纪镜头前唯一愿意配合的模特。

但越是靠近，就越能看到她们不愿展示的那一面——林澄在深夜天台独自叹气的背影，雪纪翻看一张泛黄旧照时无法抑制的颤抖。

**这个秋天到冬天的故事，没有轰轰烈烈，却有许多让人想要按下暂停键的瞬间。**

### 5.2 章节结构（每条路线）

```
┌─── 第一章：偶遇 (Day 1-3) ──────────────────────────┐
│  建立第一印象，玩家首次与角色互动。                       │
│  情感状态：好奇 / 有趣的陌生人                          │
│  关键词：意外、新鲜感、标志性特质                        │
└──────────────────────────────────────────────────────┘
           ↓
┌─── 第二章：靠近 (Day 4-10) ─────────────────────────┐
│  通过多次日常互动积累好感，找到合作/共处的契机。          │
│  情感状态：想更多了解 / 期待见面                        │
│  关键词：日常积累、共同经历、渐入佳境                    │
└──────────────────────────────────────────────────────┘
           ↓
┌─── 第三章：波澜 (Day 11-17) ────────────────────────┐
│  出现误解/矛盾/外部变量，关系受到考验。                   │
│  情感状态：不安 / 怕失去 / 自我怀疑                     │
│  关键词：冲突、距离感、第三方角色介入                     │
└──────────────────────────────────────────────────────┘
           ↓
┌─── 第四章：破冰 (Day 18-24) ────────────────────────┐
│  双方鼓起勇气打破僵局，关系质变。                        │
│  情感状态：决心 / 想要坦诚 / 不再逃避                   │
│  关键词：追上去、说出口、第一次叫名字                     │
└──────────────────────────────────────────────────────┘

│  情感确认。根据好感度和关键选择，分岔为不同结局。          │
│  情感状态：确认 / 释然 / 温暖                          │
│  关键词：告白、CG定格、未来约定                         │
└──────────────────────────────────────────────────────┘
```

## 六、角色路线详细设计

### 6.1 林澄路线（学生会线）

#### 第一章：偶遇 — 「风吹乱的不只是文件」

**核心事件 L-1-1：校门口的风**
- Day 1 / 早晨 / 校门
- 你第一天到校，校门口的风突然大了起来。林澄抱着一摞活动文件被吹得手忙脚乱，几张纸飞了出去。她皱着眉快步去追，始终没有开口找人帮忙。
- **选择**：
  - A「追上去帮她捡回飞走的文件」→ lin+3, 她愣了一下然后认真道谢 → flag: `lin_ch1_helped`
  - B「站在原地提醒她有张纸飞到身后了」→ lin+1, 她点点头自己去捡了

**核心事件 L-1-2：学生会的拜访**
- Day 2 / 放学后 / 社团楼
- 班主任建议你加入一个社团融入校园生活，你来到社团楼。学生会的门开着，林澄一个人在整理一面巨大的活动公告板。你敲了敲门框。
- **选择**：
  - A「说你是新转来的，想了解学生会」→ lin+2, 她简短但认真地介绍了学生会的工作 → flag: `lin_ch1_visited`
  - B「帮她把掉下来的图钉递过去」→ lin+2, 她终于停下来看了你一眼，说了句"你是那个新转来的？"

**日常随机 L-daily-1：值日检查** (可重复触发)
- 早晨 / 校门
- 林澄在校门口拿着签到表检查值日情况。你经过时她抬起头，随口说了句"早。"
- → lin+1（每天首次互动）

## 八、结局系统

### 8.1 结局类型总览

| 类型 | 条件 | 演出 | CG |
|------|------|------|-----|
| **真心结局 (True End)** | 达成所有关键 flag + 好感 ≥ 15 | 告白场景 + 温暖独白 + 结局画面 | ✅ 专属 CG |
| **温暖结局 (Warm End)** | 达成大部分关键 flag + 好感 ≥ 15 | 含蓄确认 + 未来约定 | ✅ 变体 CG |
| **普通结局 (Normal End)** | 好感 8-14 | 感谢与告别 + 下学期的约定 | ✅ 日常 CG |
| **遗憾结局 (Miss End)** | 好感 < 8 | 无言的离别 | ❌ 无 CG |

### 8.2 结局判定流程

```
Day 28 → 进入结局判定
    │
    ├─ lin >= 15 且 has_flag(lin_ch4_confession)?
    │   ├─ YES → has_flag(lin_ending_true)?
    │   │           ├─ TRUE: 真心结局（握手线）
    │   │           └─ WARM: 温暖结局（围巾线）
    │   └─ NO ↓
    │
    ├─ lin >= 8?
    │   ├─ YES → 普通结局
    │   └─ NO → 遗憾结局
    │
    ├─ yuki >= 15 且 has_flag(yuki_ch4_developed)?
    │   ├─ YES → has_flag(yuki_ending_true)?
    │   │           ├─ TRUE: 真心结局（面对面线）
    │   │           └─ WARM: 温暖结局（照片线）
    │   └─ NO ↓
    │
    ├─ yuki >= 8?
    │   ├─ YES → 普通结局
    │   └─ NO → 遗憾结局
    │
    └─ 两条线都未达标 → 通用遗憾结局
        「学期结束了，你回头看了一眼校园。
         银杏叶已经落光了。」
```

### 8.3 双线冲突处理

当两条路线同时满足结局条件时：
- **优先级**：触发第一个结局事件的路线优先
- **设计引导**：通过时间分配和事件密度自然分流——认真追一条线的玩家很难同时满足另一条线的所有关键 flag
- **后续版本**：可增加「后宫结局」或「友情结局」处理双高好感

## 十、好感度与数值设计

### 10.1 好感度阶段系统

| 等级 | 好感度范围 | 林澄称谓变化 | 雪纪称谓变化 | 对话语气变化 |
|------|-----------|-------------|-------------|------------|
| **陌生** | 0-2 | "同学" / "你" | "……" / 不主动说话 | 礼貌但疏远 |
| **认识** | 3-5 | "陆辰同学" | "你……" (犹豫开口) | 偶尔多说两句 |
| **熟悉** | 6-9 | "陆辰" | "陆辰" (叫名字了) | 主动搭话，偶尔开玩笑 |
| **亲近** | 10-14 | "辰……" (偶尔口误) | "我跟你说件事" | 分享秘密，表达在意 |
| **心意** | 15+ | "你啊……" (无奈又温柔) | "……只有你" | 坦白心意，特殊台词 |

### 10.2 好感度获取途径汇总

| 途径 | 单次增量 | 频率 | 理论上限 |
|------|---------|------|---------|
| 核心事件·最优选择 | +3 ~ +5 | 每个事件 1 次 | 约 40 (10个核心事件) |
| 核心事件·次优选择 | +1 ~ +3 | 每个事件 1 次 | 约 20 |
| 日常随机事件 | +1 | 每天 1-2 次 | 约 28-56 (28天) |
| 特殊物品（热可可等） | +1 ~ +2 | 剧情触发 | 约 5 |

**设计目标**：
- 不看日常、只走核心事件最优选择 → 约 40 分，轻松达到 True End (15)
- 只走核心事件次优选择 → 约 20 分，可达到 Normal End (8)
- 完全不关注一个角色 → 0-5 分，Miss End
- 日常事件是保险垫 → 漏了几个关键选择也能靠日常补回来

### 10.3 平衡目标

- **首次通关时长**：60-90 分钟（一条路线）
- **全结局收集时间**：4-6 次通关
- **单日可触发事件数**：2-4 个（含日常）
- **关键选择密度**：平均每 2-3 天一个关键抉择点
