## 一、通用规格

### 1.1 技术规格

| 项目 | 规格 |
|------|------|
| **画幅** | 512 × 1024 px |
| **格式** | PNG（透明背景） |
| **色彩空间** | sRGB |
| **位深** | 32 bit (RGBA) |
| **风格** | 日系校园 / 柔和暖色调 / 轻水彩质感 |
| **引用路径** | `res://assets/portraits/char_{id}_{expression}.png` |

### 1.2 画面构图规范

```
┌──────────────────────┐  ← 512px
│                      │
│    ┌──────────┐      │  ← 头顶留白 ≈ 80px
│    │  头部    │      │
│    │          │      │
│    ├──────────┤      │
│    │  肩部    │      │
│    │          │      │
│    ├──────────┤      │
│    │  胸部    │      │  ← 视线在画面上 1/3 处
│    │  至腰    │      │
│    │          │      │
│    ├──────────┤      │
│    │  裁切线  │      │  ← 膝盖上方裁切
│    └──────────┘      │
│                      │
└──────────────────────┘  ← 1024px
```

**构图要点**：
- 角色居中偏左（预留对话面板空间）
- 视线高度在画面上 1/3 处
- 半身像，裁切至膝盖上方
- 头顶留白约 80px（适配 UI 边距）
- 底部自然渐隐，不要硬切

### 1.3 色调基准

| 角色 | 主色调 | 辅助色 | 整体氛围 |
|------|--------|--------|---------|
| 林澄 | #4A6FA5 (深蓝灰) | #F5C6D0 (淡粉) | 干练·温柔隐藏在认真背后 |
| 雪纪 | #B8C4D8 (银灰蓝) | #F0D58C (暖金) | 安静·光影交错的距离感 |
| 苏逸 | #7B9E6C (橄榄绿) | #D4C9A8 (暖米) | 温和·即将离开的学长 |
| 小唯 | #E88B72 (珊瑚) | #6BB5C9 (浅湖蓝) | 活泼·带点锋利的直率 |
| 周老师 | #9B8EC4 (淡紫) | #E8DFD0 (暖灰) | 温婉·稳定的安全感 |

### 1.4 风格参考词

**整体画风**：清新校园 × 微水彩 × 日系轻小说插图风
- 线条：纤细但有力，不追求极致精细，留有手绘感
- 上色：大面积平涂 + 水彩风渐变阴影
- 光源：统一柔和侧光（左上 45°）
- 特征：眼睛略大但不夸张，符合写实向偏清新比例

## 二、主角立绘详细设定

### 2.1 林澄 (Lin Cheng) — ID: `lin`

#### 基本视觉

| 项目 | 描述 |
|------|------|
| **身高** | 165cm（画面中偏高） |
| **发型** | 齐肩黑色直发，自然垂落，发尾微微内卷 |
| **发色** | #1A1A2E (黑带深蓝调) |
| **瞳色** | #4A6FA5 (深蓝灰) |
| **肤色** | #FDE8D4 (自然偏暖) |
| **眼镜** | 银框方形镜片，镜腿细，鼻托处有微小反光 |
| **手表** | 左手腕简洁银色指针表，表带皮质 |
| **体型** | 标准偏瘦，肩线端正（习惯性挺背的姿态） |

#### 服装设定

**标准校服**（日常差分基础）：
- 白色长袖衬衫（烫得笔直，领口第一颗扣子总是扣上的）
- 深蓝色百褶裙（校服标准款，长度及膝）
- 白色过膝袜
- 棕色平底皮鞋
- **特征细节**：衬衫袖口处有一个小小的咖啡渍（自己没注意到的，暴露她赶时间的一面）

**冬装变体**（第五章用）：
- 在校服外加深灰色针织开衫
- 围巾：粉灰相间的格纹围巾（温暖结局 CG 关键道具）

#### 表情差分列表

| 差分名 | 文件名后缀 | 适用场景 | 描述 |
|--------|-----------|---------|------|
| **默认** | `_default` | 日常交流 | 表情平静，嘴唇微抿，目光笔直 |
| **微笑** | `_smile` | 感谢/放松时 | 嘴角微微上扬，眼睛柔和，但克制 |
| **认真** | `_serious` | 工作/交代事项 | 眉头微蹙，目光锐利，嘴巴微张（在说话） |
| **害羞** | `_shy` | 被发现甜食秘密等 | 脸颊微红，目光下移，嘴唇咬住下唇 |
| **疲惫** | `_tired` | 加班/苦撑时 | 眼神黯淡，肩膀微微下垂，嘴角绷紧 |
| **感动** | `_moved` | 烟花下/天台告白 | 眼眶微红，泪光闪烁，但在努力笑 |
| **慌张** | `_flustered` | 便当盒被发现等 | 眼睛睁大，脸颊通红，双手不知放哪 |

#### 关键视觉细节

1. **银框眼镜** — 始终保持干净无指纹（她的性格映射），唯一一次镜片起雾是在哭泣后
2. **手表** — 永远精确指向对应时段的时间，是连接她与「时间系统」的视觉符号
3. **姿态** — 默认站得笔直，双手自然垂于身侧或抱着文件夹。随着好感度提升，姿态应更放松

### 2.2 雪纪 (Yuki) — ID: `yuki`

#### 基本视觉

| 项目 | 描述 |
|------|------|
| **身高** | 158cm（画面中偏矮，显得娇小） |
| **发型** | 银灰色及腰长发，自然散落，有细微卷曲（不是直发） |
| **发色** | #B8C4D8 (银灰蓝，带微微紫光泽) |
| **瞳色** | #7A8BA8 (灰蓝，像是暴风雨前的天空) |
| **肤色** | #FAF0E6 (偏白皙) |
| **耳钉** | 右耳小巧星形银色耳钉（左耳无） |
| **体型** | 娇小纤细，肩线较窄，手指修长（常年操作相机） |

#### 服装设定

**标准搭配**：
- 白色短袖衬衫（校服但袖口稍微卷起）
- 深蓝色百褶裙（比林澄穿得略短一截）
- **卡其色工装外套**（标志性，口袋鼓鼓的——装着镜头盖、擦镜布、一块备用电池）
- 黑色短靴（方便到处跑着取景）
- 脖子上挂着一台银色微单相机（Fuji X100 风格造型）
- **特征细节**：工装外套右口袋隐约露出折叠小刀的金属柄（用来裁照片的）

**暗房造型变体**：
- 去掉外套，只穿衬衫
- 加一条深色围裙（防药水）
- 头发用一根银色发夹别在耳后

#### 表情差分列表

| 差分名 | 文件名后缀 | 适用场景 | 描述 |
|--------|-----------|---------|------|
| **默认** | `_default` | 独处/观察 | 面无表情，但不冰冷——像在思考光线 |
| **嘴角翘** | `_slight_smile` | 拍到好照片时 | 嘴角不自觉地微微翘起（她自己没意识到） |
| **专注** | `_focused` | 拍照/冲洗 | 单眼微闭（透过取景器），另一只眼专注锐利 |
| **脸红** | `_blush` | 被反拍/被夸 | 耳尖和脸颊泛红，目光别过去 |
| **空洞** | `_blank` | 与小唯争吵后 | 眼神失焦，嘴唇紧闭，像人偶 |
| **哭泣** | `_cry` | 冲洗旧胶卷 | 无声落泪，表情不扭曲——安静的泪 |
| **笑** | `_smile` | 灯塔告白 | 第一次面对镜头的真心笑容，眼睛弯成月牙 |

#### 关键视觉细节

1. **相机** — 她的保护壳，挂在脖子上时像一面盾牌。在灯塔告白时递给男主，是交出信任的象征
2. **银灰长发** — 在不同光线下呈现不同色调（暖光下带金，冷光下偏紫），象征她眼中"光"的主题
3. **右耳星形耳钉** — 只有近距离才看得到，是玩家"靠近"的视觉奖励
4. **手指** — 修长灵活，冲洗照片时的动作应有专业感
