# Campus Heartbeat — 产品路线图

> 版本: v1.0 | 维护: Patrick Star (PO) | 日期: 2026-03-20

---

## 项目现状快照

### ✅ 已完成（Sprint 0 — 骨架搭建）
- Godot 4.2 工程初始化，gl_compatibility 渲染
- 5 大 Autoload 系统（GameState / SaveManager / TimeSystem / AffectionSystem / EventSystem）
- 1 张校园地图（代码绘制占位），5 个交互点
- 2 个可攻略角色（林澄、雪纪），各 3 个事件，2 张 CG
- 完整闭环：移动→交互→对话→选择→好感→时间推进→CG解锁→存读档
- GitHub 仓库 https://github.com/DongIY/cooldiglu

### ⚠️ 主要短板
- 全部视觉为代码绘制占位（零图片资源）
- 零音频
- 无标题画面 / 设置菜单 / 结局系统
- 对话系统极简（无打字机效果、无立绘、无多页）
- 剧情量极少（约 5 分钟通关）
- 单存档槽位

## Sprint 1: 体验外壳 🎯 当前

> 目标：让玩家拥有「完整产品感」的第一印象——有标题画面进入、对话有立绘和选项动画、UI 不再是裸控件。

### 交付物清单

| ID | 任务 | 负责 | 验收标准 |
|----|------|------|---------|
| S1-01 | 标题画面场景 | cc+aa | 新游戏/继续/退出三个按钮，有背景图，继续按钮仅存档存在时可点 |
| S1-02 | 对话系统升级 — 立绘显示 | cc+dd | 对话面板左侧显示说话者立绘，从 characters.json 读取 portrait 路径 |
| S1-03 | 对话系统升级 — 打字机效果 | cc | 文字逐字显示（可配速度），点击/Enter 跳过到全显 |
| S1-04 | 对话系统升级 — 多页支持 | cc+dd | events.json 支持 pages[] 数组，自动翻页，最后一页显示选项 |
| S1-05 | UI 主题基础版 | aa | 统一配色、字体大小、按钮样式，不再用 Godot 默认灰色主题 |
| S1-06 | 角色占位立绘 | aa | 为林澄、雪纪各出 1 张简约风格立绘 SVG/PNG（512×1024） |
| S1-07 | 标题背景图 | aa | 校园场景背景图 1 张（1280×720） |

### 不做项（Scope Out）
- 不做 Tilemap 地图替换（S2）
- 不做音频系统（S4）
- 不做多存档（S5）
- 不新增剧情事件（S3）

### 依赖关系
```
S1-06, S1-07 (aa)  ──→  S1-01 (cc, 需要背景图)
                   ──→  S1-02 (cc, 需要立绘)
S1-04 (cc+dd)      ──→  需要先商定 events.json 新格式
S1-05 (aa)          ──→  独立，可并行
S1-03 (cc)          ──→  独立，可并行
```
