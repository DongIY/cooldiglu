# QA Smoke Checklist

## P0 核心闭环

- [ ] 项目可以被 Godot 4.x 正常打开
- [ ] 主场景可正常进入
- [ ] 玩家可移动且不会卡死
- [ ] 互动提示会随位置变化
- [ ] 在不同点位可触发不同事件
- [ ] 选项可改变好感度
- [ ] 时段推进后事件可发生变化
- [ ] 达成条件后能解锁 CG
- [ ] 图鉴能正确显示已解锁状态
- [ ] 存档后重新读取数据一致

## P1 易错点

- [ ] 同一事件不会被错误重复触发
- [ ] 条件不满足时不会弹出错误对话
- [ ] 不存在空角色名、空文本或缺失事件 ID
- [ ] 快速存档 / 读档不会破坏当前状态
- [ ] 没有 CG 资源时也能正常显示占位信息

## P1 引用与资源

- [ ] JSON 配置可以成功读取
- [ ] 场景引用的脚本路径有效
- [ ] Autoload 注册的脚本均可加载
- [ ] 图鉴配置中的 CG ID 与解锁逻辑一致

## P2 体验检查

- [ ] HUD 信息不拥挤
- [ ] 对话阅读节奏清晰
- [ ] 不同角色事件区分明显
- [ ] 初次进入的引导足够明确
